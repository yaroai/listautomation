# 100 Tool-Stack Video Texts (OpenSwarm)

Matches your reference reels ([reel 1](https://www.instagram.com/reel/DVNcIoBgH9Q/), [reel 2](https://www.instagram.com/reel/DV-JhhyjLdh/)) — calm, factual voice, `Tool = function. (Cost)` format. Every OpenSwarm line is now a **specific, literal function** (same style as `Claude = coding`, `Figma = UI design`), grounded in what OpenSwarm actually does. OpenSwarm appears once (occasionally twice) per list.

**OpenSwarm's real functions** (from [VRSEN/OpenSwarm](https://github.com/VRSEN/OpenSwarm) + [openswarm-ai](https://github.com/openswarm-ai/openswarm)): agent orchestration, deep research, data analysis, slide decks (→PPTX), docs (Word/PDF), image gen, video gen, and a virtual assistant with 10,000+ integrations (Gmail, Slack, GitHub, HubSpot) for outreach/messaging/scheduling. So each label below is a genuine capability, matched to the stack it sits in.

- **Format A:** `Tool = function. (Cost)`
- **Format B:** `- Tool = function. (Cost)` + a header
- **Format C:** `Tool → function` (arrow, no cost)
